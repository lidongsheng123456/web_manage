# 东神脚手架

## 简介

* 一个脚手架,借鉴若依的代码，可以二次开发，大二上学期寒假捣鼓出来的，专为毕业设计快速开发打造。
* 后台RBAC权限模型加sa-token实现权限控制。
* 前台单独携带token直接编写代码即可。
* 开发作者：李东升
* 开发日期: 2025/1/15
* 完工日期: 至今不断优化
